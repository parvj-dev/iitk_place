Job Description

 

UI/ UX Designer

 

We at VMock are looking for a talented Web Designer to create amazing user experiences. The ideal candidate should have an eye for clean and artful design, possess superior user interface design skills and be able to translate high-level requirements into interaction flows and artefacts, and transform them into beautiful, intuitive, and functional designs. Candidates with experience in HTML and CSS would be a plus. 

 

Responsibilities: 

 

  • Execute all visual design stages from concept to final hand-off to engineering 
  • Conceptualize original ideas that bring simplicity and user friendliness to complex design roadblocks 
  • Create wireframes and user flows
  • Conceptualize and design marketing and sales collaterals for Facebook and other social media 
  • Design infographics and create customized landing pages for display and online campaigns 
  • Execute creative briefs across all verticals 
  • Present and defend designs and key milestone deliverables to peers and executive level stakeholders 
  • Establish and promote design guidelines, best practices and standards 

Requirements:

 

  • Demonstrable Graphic/ Product design skills with a strong portfolio
  • Good command in Photoshop, Illustrator or other visual design and wire-framing tools 
  • Solid understanding of typography, colour presence, print layout and marketing material
  • Knowledge of any prototyping software e.g. Sketch app, Axure would be an added advantage
  • Excellent visual design skills with sensitivity to user-system interaction 
  • Ability to solve problems creatively and effectively 
  • Up-to-date with the latest Web trends, techniques, and technologies 
  • Knowledge of Agile/Scrum development process preferred
