Job Description The job holder is responsible for delivering Global Architecture Mockup and associated System engineering tasks for different aircraft programs on Time, Cost and Quality. The various activities to be performed are:
o Creation and update of Physical System architecture based upon a given Target configuration and associated input:
 Screening and Implementation of Change Request
 Convergence check of Geometry between Global Architecture Mockup and Detailed design (Digital Mockup) and necessary implementation
 Convergence check of electrical routes/function between Global Architecture Mockup and CIRCE (Treatment estimation tool)
 Check and implementation of Electrical Schematics (Functional diagram) in Global Architecture Mockup
o Verification and Validation:
 Analysis of System installation/safety requirement (SIRD) using Global Architecture Mock up
 Creating Proof of compliance to capture documentary evidence of the analysis
o Trade off studies:
 Feasibility studies for new change proposal
 Providing System Architecture/Installation input for various improvement studies
 Creation of post mod architecture
 Check of compliance status of Post Mod against Particular Risks other SIRD requirement
 Weight/cost/ performance assessment
o Request for Support Activities:
 From System design/System Installation
 For improvement of Process, Methods and Tools
 Participation to Plateaus
o Interfacing with relevant counterparts in Europe to ensure maximum effectiveness of Physical Systems Architecture design
o Project Management

Required Technical Expertise Overall Knowledge on aircraft and its Systems Basic understanding of Electrical System Installation and Integration process Good understanding of Configuration Management and Change process Working knowledge of Mechanical systems would be an added advantage
Soft Skills Teamwork Good communication skills Willingness to learn new skills, self-development Being proactive in initiating Process/Product improvement
