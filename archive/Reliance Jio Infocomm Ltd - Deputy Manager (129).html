Description :

1. Creating Specifications and Test Cases for 5G NR gNodeB: This includes following:

a) In-depth Study of 3GPP specifications for 5G NR

b) Gain understanding of existing LTE eNodeB specifications and how it will interwork with 5G NR gNB in Non Stand alone Mode or EN-DC mode.

c) Study and understand 5G gNode RAN Architecture Options

d) Create detailed specifations for all Layers and covering all aspects of gNodeB.

e) Creating Protocol Specifications for all Interfaces of 5G RAN architecture

f) Create Test Cases for each Layer of 5G Protocol Stack - PHY, MAC, RLC, PDCP, Interworking with LTE etc.

2. Perform 5G NR Link Level Simulations in MATLAB/Simulink. This includes:

a) Creation of signal processing chains for Physical Layer (PHY) for various PHY channels PRACH, PUSCH, PUCCH for Uplink and PDSCH, PDCCH and others for Downlink.

b) Create Receiver Algorithms such as Channel Estimation, Frequency Error Estimation, Noise Variance Estimation etc. to model gNB receiver and verify its performance through simulations

c) Create Downlink Beam froming algorithms and simulate the same.

d) Create algorithms for Link Adaptation or Adaptive Modulation and Coding and simulate the same in Matlab or other tools

3. Feature Validation, Performance Evaluation and Inter-operability Testing of 5G NR gNodeBs. This includes:

a) Thorough validation of all Protocol Layers in Simulated Environment as well as in end products with Real 5G devices and 5G core network.

b) Testing of gNodeB performance by injecting simulated load of 100s of devices and lakhs of Connection & Handover Attempts or load of real devices.

c) Testing of Interoperability between gNB and eNodeB, gNodeB and Core and gNodeB and device to ensure seamless interworking.
